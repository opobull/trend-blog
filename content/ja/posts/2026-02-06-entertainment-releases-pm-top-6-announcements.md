## 1. 2026年冬季五輪開会式、今夜ミラノで開幕

![冬季五輪ミラノ2026](/images/entertainment-pm-2026-02-06.jpg)

**2026年ミラノ・コルティナ冬季オリンピック**が今夜、ミラノの象徴的な**サン・シーロ・スタジアム**で壮大なセレモニーと共に開幕します。「**アルモニア（調和）**」をテーマにした開会式は、**2026年2月6日**現地時間午後8時（米国PT午前11時/ET午後2時）に始まります。

**マライア・キャリー**と**アンドレア・ボチェッリ**が音楽パフォーマンスのヘッドライナーを務め、キャリーはドメニコ・モドゥーニョの名曲「Nel blu, dipinto di blu」（通称「ヴォラーレ」）を披露する予定です。その他、イタリアのオペラ歌手**チェチーリア・バルトリ**、歌手**ラウラ・パウジーニ**、中国人ピアニスト**ラン・ラン**、ラッパー**ガリ**も出演します。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| イベント | 開会式 |
| 会場 | サン・シーロ・スタジアム、ミラノ |
| テーマ | 「アルモニア」（調和） |
| 開始時間 | 中央欧州時間 午後8時（2月6日） |
| 放送 | NBC、Peacock（米国） |
| ヘッドライナー | マライア・キャリー、アンドレア・ボチェッリ |

セレモニーには**3,500人の選手**、**1,300人以上のパフォーマー**、182のオリジナルデザイン、1,400着以上の衣装が登場。イベントプロデューサーのマルコ・バリッチは「メイド・イン・イタリー」の最高峰を祝い、「シンプルさと本物らしさ」を重視したショーを約束しています。

ユニークな特徴として、ミラノ（平和の門）とコルティナ・ダンペッツォで**2つの聖火台**が同時に点火され、どちらもレオナルド・ダ・ヴィンチへのオマージュとしてデザインされています。

注目の出席者には、**トム・クルーズ**（LA2028アンバサダーとして参加）、**スヌープ・ドッグ**（NBC特派員として再登場）、米副大統領**JD・ヴァンス**、イタリア首相**ジョルジャ・メローニ**が含まれます。

## 2. A24が恐怖の新予告編公開「undertone」- "これまでで最も怖い映画"

A24が**undertone**のセカンド予告編を公開しました。すでに「容赦なく怖い」と評され、2026年ホラー作品の最有力候補とされる心理ホラー映画です。**2026年3月13日**劇場公開予定。

新人監督**イアン・トゥアソン**の脚本・監督作品で、undertoneは超常現象をテーマにしたポッドキャストを友人**ジャスティン**（The White Lotusのアダム・ディマルコ）と共同ホストする**エヴィ**（The Handmaid's Taleのニーナ・キリ）を描きます。瀕死の母の介護のため実家に戻ったエヴィは、超常現象を調査する中で、悪魔的な力への扉を開いてしまいます。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| スタジオ | A24 |
| 公開日 | 2026年3月13日 |
| 監督 | イアン・トゥアソン（デビュー作） |
| 出演 | ニーナ・キリ、アダム・ディマルコ |
| 製作費 | 50万ドル |

本作のユニークな特徴は、**音がホラーの主要な要素**であること。エヴィが録音から低周波のオーディオ周波数を拾い、超自然的なささやきが明らかになります。批評家たちはすでにブライアン・デ・パルマの「ブロウアウト」と比較し、ニーナ・キリのほぼソロ演技を絶賛しています。

サンダンス2026での上映後、トゥアソンはAtomic MonsterとBlumhouseのために**パラノーマル・アクティビティ・フランチャイズの復活**を任されました。Colliderの早期レビューでは、undertoneは**2027年アカデミー賞の音響賞**の候補として検討されるべきだと示唆しています。

## 4. Prime Video「Cross」シーズン2、マシュー・リラードも出演で2月11日配信開始

**Crossシーズン2**が**2026年2月11日**にPrime Videoで配信開始。**ジェイムズ・パターソン**のアレックス・クロス小説シリーズを原作とした人気刑事ドラマの続編です。3話同時配信でスタートし、その後毎週配信、最終話は3月18日。

**アルディス・ホッジ**が法医学心理学者兼刑事**アレックス・クロス**役で復帰。アイザイア・ムスタファ、アローナ・タル、サマンサ・ウォークス、フアニータ・ジェニングス、ケイレブ・イライジャ、メロディ・ハード、ジョニー・レイ・ギルも再登場します。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| プラットフォーム | Prime Video |
| 配信日 | 2026年2月11日 |
| エピソード数 | 8話（初回3話、以降毎週） |
| 最終話 | 2026年3月18日 |
| 主演 | アルディス・ホッジ |

新キャストとして**マシュー・リラード**（Five Nights at Freddy's、スクリーム）が未公開の役で参加し、今シーズンのダークな展開への期待が高まっています。ホッジは「誰も安全ではない」とシーズン2をほのめかし、変化する忠誠心と曖昧な道徳性を予告しています。

Crossは復帰を前に静かにPrime Videoのストリーミングチャートを上昇中。現在オーストラリア（10位）、カナダ（9位）、ナイジェリア（7位）でトップ10入りしています。

## 6. Netflix「Love Is Blind」シーズン10、2月11日からオハイオを舞台に配信

**Love Is Blindシーズン10**が**2026年2月11日**にNetflixで配信開始。恋愛リアリティ番組の記念すべき第10弾は**オハイオ州**（コロンバス周辺地域）が舞台です。

**ニック＆ヴァネッサ・ラシェイ**がホストを務め、シリーズ独自のフォーマットを継続。参加者たちは相手の姿を見ることなく婚約し、「ポッド」を通じてのみコミュニケーションを取った後、対面し、最終的には結婚式に臨む可能性があります。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| プラットフォーム | Netflix |
| 配信日 | 2026年2月11日 |
| シーズン | 10 |
| ロケ地 | オハイオ州（コロンバス周辺） |
| ホスト | ニック＆ヴァネッサ・ラシェイ |
| 初回エピソード数 | 6話 |

初回は6話同時配信で、その後シーズンが結婚式のフィナーレに向けて進むにつれて毎週追加エピソードが公開されます。

関連ニュースとして、**Love Is Blind: Netherlands**の制作も発表され、オランダのセレブカップル**ニコレット・ファン・ダム**と**バス・スミット**がホストを務めることが確認されました。

シーズン10は、数多くの国際版を生み出し、Netflixの最も視聴された脚本なしコンテンツの上位にランクインし続けている大人気リアリティフランチャイズの重要なマイルストーンとなります。
