## 1. エラゴン Disney+シリーズがショーランナーと制作チームを発表

![Eragon Disney+](/images/entertainment-am-2026-02-06-1.jpg)

Disney+の実写版**エラゴン**シリーズが、制作チームの発表で大きく前進しました。**トッド・ハーサン**（ハイ・ポテンシャル）と**トッド・ヘルビング**（スーパーマン＆ロイス）が共同ショーランナーとして参加し、ハーサンは原作者クリストファー・パオリーニと共に共同クリエイターを務めます。

監督の**マーク・ウェブ**（アメイジング・スパイダーマン、500日のサマー）もエグゼクティブ・プロデューサーとして参加。シリーズは20thテレビジョンをスタジオとしてDisney+で開発中です。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| プラットフォーム | Disney+ |
| ステータス | 開発中 |
| ショーランナー | トッド・ハーサン、トッド・ヘルビング |
| EP/監督 | マーク・ウェブ |
| 共同クリエイター | クリストファー・パオリーニ（原作者） |

**公式あらすじ：**「運命により100年以上ぶりの最初のドラゴン・ライダーに選ばれた平凡なティーンエイジャーは、ドラゴンと揺るぎない絆を築き、古代魔法を習得し、ライダーを滅ぼした狂気の王に挑まなければならない。」

「遺産の継承者」シリーズは世界中で4000万部以上を売り上げています。この作品は、2006年の酷評された映画以来の新たなスタートとなり、ファンは2021年から#EragonRemakeハッシュタグでリメイクを求めてきました。

## 2. ケープ・フィアー - Apple TVがスコセッシ＆スピルバーグ制作スリラーの6月プレミアを発表

![Cape Fear Apple TV](/images/entertainment-am-2026-02-06-1.jpg)

Apple TV+の10話サイコスリラー**ケープ・フィアー**が、**2026年6月5日**に最初の2話でプレミア公開されます。以降は7月31日まで毎週金曜日に新エピソードが配信されます。

シリーズには**エイミー・アダムス**、**ハビエル・バルデム**、**パトリック・ウィルソン**が出演し、クラシックスリラーの現代版リメイクとなります。マーティン・スコセッシとスティーヴン・スピルバーグがエグゼクティブ・プロデューサーを務め、ニック・アントスカがクリエイター兼ショーランナーです。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| 公開日 | 2026年6月5日 |
| プラットフォーム | Apple TV+ |
| エピソード数 | 10話 |
| 配信スケジュール | 初回2話、以降週1話 |
| EP | マーティン・スコセッシ、スティーヴン・スピルバーグ |

**あらすじ：** 幸せに結婚した弁護士アンナ（アダムス）とトム・ボウデン（ウィルソン）に嵐が迫る。彼らが投獄に追い込んだ悪名高い殺人犯マックス・ケイディ（バルデム）が出所し、復讐を望んでいる。

このシリーズは「緊迫したヒッチコック的スリラーであり、21世紀のアメリカの実録犯罪への執着を検証する」と紹介されています。

## 3. モナーク: レガシー・オブ・モンスターズ シーズン2が新タイタンを公開

![Monarch Legacy of Monsters](/images/entertainment-am-2026-02-06-1.jpg)

Apple TV+は**モナーク: レガシー・オブ・モンスターズ シーズン2**の公式トレーラーを公開し、恐ろしい新たな脅威**タイタンX**を明らかにしました。これは怪獣の階層を脅かす巨大な触手を持つ生物です。

シリーズは**2026年2月27日**に復帰し、モンスターバースの拡大するテレビユニバースでゴジラ、コング、そしてこの新たな海底の脅威が登場します。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| 公開日 | 2026年2月27日 |
| プラットフォーム | Apple TV+ |
| シーズン | 2 |
| 新たな脅威 | タイタンX（触手を持つ怪獣） |

トレーラーは、チームが最新の騒乱を引き起こしている怪獣がコングでもゴジラでもなく、さらに大きな何かであることに気づくシーンから始まります。この生物は巨大な触手を使って有人船を海底へ引きずり込みます。

## 5. ブリリアント・マインズ - NBCが医療ドラマをスケジュールから削除

![Brilliant Minds NBC](/images/entertainment-am-2026-02-06-1.jpg)

ザカリー・クイント主演のシリーズにとって大きな打撃として、**NBCはブリリアント・マインズをシーズン2の14話後にスケジュールから削除**しました。残りの6話は今シーズン後半または夏に未定の日程で放送されます。

この措置は、2026年冬季オリンピック終了後の2月23日プレミアから月曜日に2時間枠で放送される**ザ・ヴォイス シーズン29**のためのスペースを確保するものです。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| ネットワーク | NBC |
| ステータス | スケジュールから削除 |
| 放送済みエピソード | 14話（シーズン2） |
| 残りエピソード | 6話 |
| 代替番組 | ザ・ヴォイス シーズン29 |

神経科学者オリバー・ウルフ博士を演じるクイント主演の医療ドラマは、控えめな視聴率でプレミアし、月曜夜の時間枠で視聴者を獲得するのに苦戦してきました。

## 6. ユア・フレンズ＆ネイバーズ - Apple TV+がシーズン2前にシーズン3を更新

![Your Friends and Neighbors](/images/entertainment-am-2026-02-06-1.jpg)

Apple TV+は、**2026年4月3日**のシーズン2プレミアに先立ち、ジョナサン・トロッパー制作の**ユア・フレンズ＆ネイバーズ**をシーズン3に更新しました。

**主要情報：**

| 項目 | 詳細 |
|------|------|
| プラットフォーム | Apple TV+ |
| ステータス | シーズン3更新 |
| シーズン2プレミア | 2026年4月3日 |
| クリエイター | ジョナサン・トロッパー |

このシリーズでは、ホワイトカラーの仕事を解雇されたアンドリュー「クープ」クーパー（ジョン・ハム）が予期せぬキャリアチェンジを余儀なくされます—コネチカット近郊の裕福な家を狙う泥棒になるのです。シーズン2では彼を暴露しようとする新しい隣人が登場し、ベストメント・ビレッジに新たな緊張をもたらします。
